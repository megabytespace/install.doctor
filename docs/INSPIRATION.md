# Inspiration

Projects, tools, and resources that have inspired or influenced Install Doctor's design and features.

## Provisioning and Dotfile Managers

- [MikeMcQuaid/strap](https://github.com/MikeMcQuaid/strap) - A script to bootstrap a macOS development system
- [Chezmoi](https://www.chezmoi.io/) - Cross-platform dotfile manager used by Install Doctor
- [Homebrew](https://brew.sh/) - The package manager for macOS (and Linux) that Install Doctor relies on heavily

## CLI Tools and Libraries

- [Tauri](https://github.com/tauri-apps/tauri) - Build cross-platform desktop apps with web frontends
- [Ink](https://github.com/vadimdemedes/ink) - React-based CLI rendering library
- [Cliffy](https://cliffy.io/) - Deno CLI framework
- [Lowdb](https://github.com/typicode/lowdb) - Lightweight JSON database for Node.js
- [Execa](https://github.com/sindresorhus/execa) - Better child process execution for Node.js

## Web and Self-Hosted Services

- [Coder](https://github.com/coder/coder) - Remote development environments on your infrastructure
- [Ravada](https://github.com/UPC/ravada) - Virtual desktop infrastructure manager
- [Payload CMS](https://github.com/payloadcms/payload) - Headless CMS and application framework
- [RustDesk Server](https://github.com/rustdesk/rustdesk-server) - Self-hosted remote desktop server
- [ToolJet](https://github.com/ToolJet/ToolJet) - Low-code platform for building internal tools
- [Plane](https://github.com/makeplane/plane) - Open-source project management (Jira alternative)
- [Listmonk](https://github.com/knadh/listmonk) - Self-hosted newsletter and mailing list manager
- [Answer](https://github.com/answerdev/answer) - Self-hosted Q&A platform
- [Kutt](https://github.com/thedevs-network/kutt) - Self-hosted URL shortener
- [Judge0](https://github.com/judge0/judge0) - Online code execution system
- [Dogbin](https://github.com/dogbin/dogbin) - Pastebin alternative
- [Tolgee](https://github.com/tolgee/tolgee-platform) - Open-source localization platform
- [Sorry Cypress](https://github.com/sorry-cypress/sorry-cypress) - Open-source Cypress dashboard

## Serverless and Edge

- [Serverless Cloudflare Workers](https://github.com/cloudflare/serverless-cloudflare-workers) - Serverless framework plugin for Cloudflare Workers
- [Supaflare](https://github.com/supaflare/supaflare) - URL shortener built on Cloudflare Workers and KV

## Useful Online Tools

- [Send](https://send.vis.ee/) - Encrypted file sharing
- [Rustpad](https://rustpad.io/) - Collaborative text editor
- [AnonAddy](https://anonaddy.com/) - Anonymous email forwarding
- [Claper](https://claper.co/) - Interactive presentation tool
- [Namelix](https://namelix.com/) - AI-powered business name generator
- [CyberChef](https://gchq.github.io/CyberChef) - Data encoding, encryption, and analysis toolkit

## Documentation and AI

- [Nextra](https://nextra.site/) - Next.js documentation framework
- [Next.js AI Chatbot](https://github.com/vercel-labs/ai-chatbot) - AI chatbot template
- [Isomorphic Git](https://github.com/isomorphic-git/isomorphic-git) - Pure JavaScript git implementation
- [Apostrophe CMS](https://github.com/apostrophecms/apostrophe) - Full-featured content management system

## File Sharing and Storage

- [Uppy](https://uppy.io/) - Modular file uploader for web applications
- [WebContainers](https://webcontainers.io/) - Run Node.js in the browser
- [EnvShare](https://envshare.dev/) - Share environment variables securely
